## Fonter
>Olika fonter som kan vara bra att ha som en developer.

| Fonter  | Beskrivning |
| ------------- | ------------- |
|[Google Fonts](https://fonts.google.com/)| Google fonts erbjuder på 1359 olika font familjer.
